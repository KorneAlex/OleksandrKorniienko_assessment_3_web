## Run Locally

Clone the project

```bash
  git clone https://github.com/KorneAlex/OleksandrKorniienko_assessment_3_web.git
```

Go to the project directory

```bash
  cd OleksandrKorniienko_assessment_3_web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
